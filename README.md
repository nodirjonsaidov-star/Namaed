# Task Menejer

Bu loyiha oddiy statik sayt bo‘lib, Vercel'da juda tez deploy bo‘ladi.

## GitHub ga yuklash
1. GitHub'da yangi repo oching.
2. Shu zip ichidagi fayllarni repo ichiga yuklang.
3. Commit qiling.

## Vercel ga ulash
1. Vercel'da `Connect Git Repository` ni bosing.
2. Repo'ni tanlang.
3. `Deploy` ni bosing.

Hech qanday build sozlamasi kerak emas.
